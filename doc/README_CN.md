# 两阶段并行共识

bcos-pbft是[FISCO BCOS 3.0](https://github.com/FISCO-BCOS/FISCO-BCOS)的PBFT共识算法实现，bcos-pbft设计了两阶段并行拜占庭共识，将区块链系统的共识划分成区块批量并行排序共识和区块执行结果流水线共识两个阶段，两阶段可并行进行，旨在提升区块链系统共识性能。
